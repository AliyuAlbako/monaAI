# monaAI
Mona AI Backend is a FastAPI-powered conversational AI backend built with N-ATLaS-LLM. It provides a multilingual Nigerian business assistant capable of responding in English, Pidgin, Hausa, Yoruba, and Igbo. The API supports customer chat, business inquiries, and is designed for easy integration with web and mobile frontends.

Mona AI – Multilingual Nigerian Business Assistant

Mona AI is a FastAPI-based backend service that powers a local, lightweight conversational AI using llama-cpp-python. The system is designed to act as a Nigerian-focused business assistant, capable of understanding and responding in multiple Nigerian languages including English, Pidgin, Hausa, Yoruba, and Igbo.
